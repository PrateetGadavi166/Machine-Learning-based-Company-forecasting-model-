# Machine Learning-based Company Forecasting Model 📊

A predictive analytics project that forecasts company finances using historical data and machine learning models. The project also includes interactive visualizations for easier interpretation.  

## Features ✨
- Predict future company finances with high accuracy  
- Visualize financial trends and projections  
- User-friendly interface with real-time interaction via GUI  

## Project Structure 📁
```bash
company-forecasting-model/
├── data/ # Historical company financial datasets
├── notebooks/ # Jupyter notebooks for EDA and modeling
├── models/ # Trained machine learning models
├── src/ # Source code for training and predictions
├── app/ # GUI application (Streamlit)
├── requirements.txt # Python dependencies
└── README.md # Project documentation
```

## Installation & Setup ⚙️

### 1. Clone the repository
```bash
git clone <your-repo-url>
cd company-forecasting-model
```
### 2. Create a virtual environment
```bash
python -m venv env
source env/bin/activate   # Linux/Mac
env\Scripts\activate      # Windows
```
### 3. Install dependencies
```bash
pip install -r requirements.txt
```
### 4. Run the GUI
```bash
streamlit run app/main.py
Open the link provided by Streamlit in your browser to interact with the application.
```
